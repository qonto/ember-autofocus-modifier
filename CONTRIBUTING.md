# How To Contribute

## Installation

* `git clone <repository-url>`
* `cd ember-autofocus-modifier`
* `pnpm install`

## Linting

* `pnpm lint`
* `pnpm lint:fix`

## Building the addon

* `cd ember-autofocus-modifier`
* `pnpm build`
